Overview & Scope

Spot is a Discord bot that was made for a private server of 60+ users. It was made using Python with the help of the discord.py library. The bot has an interactive, user-friendly interface which explains how to use it's many features.

  • The project is approximately 1500 lines of code.
  • You can create polls with up to 9 different options, and votes update in real-time as users vote.
  • Poll results will show in real-time as users vote.
  • The bot will announce birthdays at midnight, every night (if any).

Design custom polls, showcase unique profiles, and celebrate birthdays with your community.

Server Commands:

  • info: get detailed info about the server.
  • setlogs: set the logs channel.
  • jail: view users who are currently banned from the server.
  • boosters: view members who are currently boosting the server.
  • ping: view the current latency of the bot.

Moderation Commands:

  • kick: kick the mentioned user from the server.
  • ban: ban the mentioned user from the server.
  • unban: unban the mentioned user from the server.
  • clear: clear 'x' amount of messages above you in the given channel.

Profile Commands:

  • profile: view your profile (or the mentioned user's profile).
  • editprofile: edit sections of your profile.
  • createprofile: create your profile.

Quote Commands:

  • quote: attach an image and mention a user to add the image to the user's quotes profile.
  • quotes: view your quotes (or the mentioned user's quotes).
  • delquote: delete a quote from a user's profile.

Action Commands:

  • hug: send a virtual hug to the mentioned user.
  • slap: send a virtual slap to the mentioned user.

Misc Commands:

  • birthdays/bdays: view a list of all the server birthdays.
  • zodiacs: get a list of zodiac signs and their corresponding date ranges.
  • avatar/pfp: view your Discord avatar (or the mentioned user's avatar).
  • id: view your Discord account ID (or the mentioned user's ID).
  • fix: someone send a broken image? Use this to try and fix it.
